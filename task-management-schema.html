🔗 Relationships

team_members.team_id → teams.id - Team membership
team_members.user_id → users.id - User in team
projects.team_id → teams.id - Project belongs to team
projects.created_by → users.id - Project creator
tasks.team_id → teams.id - Task belongs to team (always)
tasks.project_id → projects.id - Task in project (NULL for adhoc tasks)
tasks.assigned_to → users.id - Task assignee
tasks.created_by → users.id - Task creator

⚡ Performance Indexes

idx_team_members_user ON team_members(user_id)
idx_projects_team ON projects(team_id)
idx_tasks_team ON tasks(team_id)
idx_tasks_project ON tasks(project_id)
idx_tasks_assigned ON tasks(assigned_to)
idx_tasks_status ON tasks(status)
idx_tasks_due_date ON tasks(due_date)
